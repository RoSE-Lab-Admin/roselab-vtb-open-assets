# roselab-vtb-open-assets

A collection of open-source assets for **lunar virtual testbeds** and **digital twins**, developed by the Robotic Space Exploration (RoSE) Lab at Colorado School of Mines.

---

## 🚀 Overview

This repository provides reusable assets to support the development, simulation, and evaluation of lunar robotics systems. These resources are intended for:

- Virtual testbeds for space robotics  
- Digital twin environments  
- Simulation-based validation and benchmarking  
- Research and educational use  

The goal is to accelerate development and reproducibility in lunar exploration research.

---

## 📦 Contents

This repository may include (depending on updates):

- 🌕 Lunar terrain models  
- 🤖 Robot models and configurations  
- 🧩 Simulation assets (URDF, meshes, environments)  
- ⚙️ Configuration files for simulators  
- 📊 Benchmarking or evaluation assets  

> **Note:** Structure and contents may evolve as the project grows.



## 🛠️ Usage
1. Clone the repository:
```bash
   git clone https://github.com/RoSE-Lab-Admin/roselab-vtb-open-assets.git
```
2. Integrate assets into your simulation framework (e.g., ROS, Gazebo, Isaac Sim, etc.)
3. Refer to individual folders for specific instructions or dependencies.
